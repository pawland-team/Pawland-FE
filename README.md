## 💻 프로젝트 실행법
1. 프로젝트를 clone 하세요.
   ```
   git clone https://github.com/pawland-team/Pawland-FE.git
   ```
2. NPM 패키지들을 설치하세요.
   ```
   npm install
   ```
3. 프로젝트를 실행하세요.
   ```js
   npm run dev
   ```
